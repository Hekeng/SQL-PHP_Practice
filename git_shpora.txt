git status	Показывает, какие файлы изменены и что «видит» Гит.	Когда запутался и хочешь понять, что происходит.
git add .	Готовит все измененные файлы к сохранению.	Перед коммитом.
git commit -m "..."	Записывает изменения в историю (локально).	Когда закончил логический кусок кода.
git push origin master	Выталкивает твои коммиты в интернет.	Сразу после коммита.
git pull origin master	Втягивает изменения из интернета и вклеивает в код.	Перед началом работы.
git push --force	«Я здесь главный!». Затирает облако твоим кодом.	Только если всё сломалось и ты уверен в своем коде.
